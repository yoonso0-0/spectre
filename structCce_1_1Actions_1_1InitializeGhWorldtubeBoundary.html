Detailed Description

template<typename BoundaryCommunicationTags>
struct Cce::Actions::InitializeGhWorldtubeBoundary< BoundaryCommunicationTags >

Initializes a GhWorldtubeBoundary.

Details

Uses:

  • initialization tag Cce::InitializationTags::GhWorldtubeBoundaryDataManager,
  • const global cache tags InitializationTags::LMax, InitializationTags::ExtractionRadius.

Databox changes:

  • Adds:
    • Tags::Variables<typename Metavariables::cce_boundary_communication_tags>
    • Tags::GhInterfaceManager (cloned from InitializationTags::GhInterfaceManager)
  • Removes: nothing
  • Modifies: nothing
Note
This action relies on the SetupDataBox aggregated initialization mechanism, so Actions::SetupDataBox must be present in the Initialization phase action list prior to this action.
