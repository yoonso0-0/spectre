20 /*!
21  * \ingroup LinearSolverGroup
22  * \brief A conjugate gradient solver for linear systems of equations \f$Ax=b\f$
23  * where the operator \f$A\f$ is symmetric.
24  *
25  * \details The only operation we need to supply to the algorithm is the
26  * result of the operation \f$A(p)\f$ (see \ref LinearSolverGroup) that in the
27  * case of the conjugate gradient algorithm must be symmetric. Each step of the
28  * algorithm expects that \f$A(p)\f$ is computed and stored in the DataBox as
29  * %db::add_tag_prefix<LinearSolver::Tags::OperatorAppliedTo,
30  * db::add_tag_prefix<LinearSolver::Tags::Operand, FieldsTag>>. To perform a
31  * solve, add the `solve` action list to an array parallel component. Pass the
32  * actions that compute \f$A(q)\f$, as well as any further actions you wish to
33  * run in each step of the algorithm, as the first template parameter to
34  * `solve`. If you add the `solve` action list multiple times, use the second
35  * template parameter to label each solve with a different type.
36  *
37  * Note that the operand \f$p\f$ for which \f$A(p)\f$ needs to be computed is
38  * not the field \f$x\f$ we are solving for but
39  * `db::add_tag_prefix<LinearSolver::Tags::Operand, FieldsTag>`. This field is
40  * initially set to the residual \f$r = b - A(x_0)\f$ where \f$x_0\f$ is the
41  * initial value of the `FieldsTag`.
42  *
43  * When the algorithm step is performed after the operator action \f$A(p)\f$ has
44  * been computed and stored in the DataBox, the conjugate gradient algorithm
45  * implemented here will converge the field \f$x\f$ towards the solution and
46  * update the operand \f$p\f$ in the process. This requires two reductions over
47  * all elements that are received by a `ResidualMonitor` singleton parallel
48  * component, processed, and then broadcast back to all elements. The actions
49  * are implemented in the `cg::detail` namespace and constitute the full
50  * algorithm in the following order:
51  * 1. `PerformStep` (on elements): Compute the inner product \f$\langle p,
52  * A(p)\rangle\f$ and reduce.
53  * 2. `ComputeAlpha` (on `ResidualMonitor`): Compute
54  * \f$\alpha=\frac{r^2}{\langle p, A(p)\rangle}\f$ and broadcast.
55  * 3. `UpdateFieldValues` (on elements): Update \f$x\f$ and \f$r\f$, then
56  * compute the inner product \f$\langle r, r\rangle\f$ and reduce to find the
57  * new \f$r^2\f$.
58  * 4. `UpdateResidual` (on `ResidualMonitor`): Store the new \f$r^2\f$ and
59  * broadcast the ratio of the new and old \f$r^2\f$, as well as a termination
60  * flag if the `Convergence::Tags::Criteria` are met.
61  * 5. `UpdateOperand` (on elements): Update \f$p\f$.
62  *
63  * \see Gmres for a linear solver that can invert nonsymmetric operators
64  * \f$A\f$.
65  */
