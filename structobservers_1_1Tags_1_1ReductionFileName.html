Detailed Description

The name of the HDF5 file on disk into which reduction data is written.

By reduction data we mean any data that is written once across all nodes. For example, an integral over the entire (or a subset of the) domain is considered reduction data, while data on a 2d surface written from a 3d simulation is considered volume data.
