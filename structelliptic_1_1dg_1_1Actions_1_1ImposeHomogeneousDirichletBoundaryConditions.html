Detailed Description

template<typename VariablesTag, typename DirichletTags>
struct elliptic::dg::Actions::ImposeHomogeneousDirichletBoundaryConditions< VariablesTag, DirichletTags >

Set field data on external boundaries so that they represent homogeneous (zero) Dirichlet boundary conditions.

This action imposes homogeneous boundary conditions on all DirichletTags

See also
elliptic::dg::homogeneous_dirichlet_boundary_conditions
Note
We cannot impose inhomogeneous boundary conditions here because it would break linearity of the DG operator: If in the system of equations \(A(x)=b\) the DG operator \(A\) had non-zero boundary contributions then \(A(x=0)\neq 0\), which breaks linearity. Instead, inhomogeneous boundary conditions are handled as contributions to the source \(b\) during initialization.

With:

  • interior<Tag> = Tags::Interface<Tags::BoundaryDirectionsInterior<volume_dim>, Tag>
  • exterior<Tag> = Tags::Interface<Tags::BoundaryDirectionsExterior<volume_dim>, Tag>
