39  /*!
40  * Return by pointer the values of the spin-weighted spherical harmonic
41  * evaluated at `theta` and `phi`.
42  *
43  * \details The additional values `sin_theta_over_2` and `cos_theta_over_2`,
44  * representing \f$\sin(\theta/2)\f$ and \f$\cos(\theta/2)\f$ are taken as
45  * required input to improve the speed of evaluation when called more than
46  * once.
47  *
48  * The formula we evaluate (with various prefactors precomputed, cached, and
49  * optimized from the factorials) is \cite Goldberg1966uu
50  *
51  * \f{align*}{
52  * {}_s Y_{l m} = (-1)^m \sqrt{\frac{(l + m)! (l-m)! (2l + 1)}
53  * {4 \pi (l + s)! (l - s)!}} \sin^{2 l}(\theta / 2)
54  * \sum_{r = 0}^{l - s} {l - s \choose r} {l + s \choose r + s - m}
55  * (-1)^{l - r - s} e^{i m \phi} \cot^{2 r + s - m}(\theta / 2).
56  * \f}
57  */

89 /*!
90  * \ingroup SwshGroup
91  * \brief Performs interpolation for spin-weighted spherical harmonics by
92  * taking advantage of the Clenshaw method of expanding recurrence relations.
93  *
94  * \details During construction, we cache several functions of the target
95  * interpolation points that will be used during the Clenshaw evaluation.
96  * A new `SwshInterpolator` object must be created for each new set of target
97  * points, but the member function `SwshInterpolator::interpolate()` may be
98  * called on several different coefficients or collocation sets, and of
99  * different spin-weights.
100  *
101  * Recurrence constants
102  * --------------------
103  * This utility obtains the Clenshaw interpolation constants from a
104  * `StaticCache`, so that 'universal' quantities can be calculated only once per
105  * execution and re-used on each interpolation.
106  *
107  * We evaluate the recurrence coefficients \f$\alpha_l^{(a,b)}\f$ and
108  * \f$\beta_l^{(a,b)}\f$, where \f$a = |s + m|\f$, \f$b = |s - m|\f$, and
109  *
110  * \f[
111  * {}_sY_{l, m}(\theta, \phi) = \alpha_l^{(a,b)}(\theta) {}_s Y_{l-1, m}
112  * + \beta_l^{(a, b)} {}_s Y_{l - 2, m}(\theta, \phi)
113  * \f]
114  *
115  * The core Clenshaw recurrence is in the\f$l\f$ modes of the
116  * spin-weighted spherical harmonic. For a set of modes \f$a_l^{(a,b)}\f$ at
117  * fixed \f$m\f$, the function value is evaluated by the recurrence:
118  *
119  * \f{align*}{
120  * y^{(a, b)}_{l_\text{max} + 2} &= y^{(a, b)}_{l_\text{max} + 1} = 0 \\
121  * y^{(a, b)}_{l}(\theta) &= \alpha_{l + 1}^{(a, b)} y^{(a, b)}_{l +
122  * 1}(\theta)
123  * + \beta_{l + 2}^{(a,b)} y^{(a, b)}_{l + 2}(\theta) + a_l^{(a, b)} \\
124  * f_m(\theta, \phi) &= \beta_{l_{\text{min}} + 2}
125  * {}_s Y_{l_{\text{min}}, m}(\theta, \phi) y^{(a, b)}_2(\theta) +
126  * {}_s Y_{l_{\text{min}} + 1, m}(\theta, \phi) y^{(a, b)}_1(\theta) +
127  * a_0^{(a, b)} {}_s Y_{l_{\text{min}}, m}(\theta, \phi),
128  * \f}
129  *
130  * where \f$l_{\text{min}} = \max(|m|, |s|)\f$ is the lowest nonvanishing
131  * \f$l\f$ mode for a given \f$m\f$ and \f$s\f$.
132  *
133  * The coefficients to cache are inferred from a mechanical but lengthy
134  * calculation involving the recurrence relation for the Jacobi polynomials.
135  * The result is:
136  *
137  * \f{align*}{
138  * \alpha_l^{(a, b)} &= \frac{\sqrt{(2l + 1)(2l - 1)}}
139  * {2\sqrt{(l + k)(l + k + a + b)(l + k + a)(l + k + b)}}
140  * \left[2 l \cos(\theta) +
141  * \frac{a^2 - b^2}{2 l - 2}\right] \\
142  * \beta_l^{(a, b)} & = - \sqrt{\frac{(2 l + 1)(l + k + a - 1)(l + k + b - 1)
143  * (l + k - 1)(l + k + a + b - 1)}
144  * {(2l - 3)(l + k)(l + k + a + b)(l + k + a)(l + k + b)}}
145  * \frac{2 l}{2 l - 2},
146  * \f}
147  * where \f$k = - (a + b)/2\f$ (which is always integral due to the properties
148  * of \f$a\f$ and \f$b\f$). Note that because the values of \f$\alpha\f$ and
149  * \f$\beta\f$ in the recurrence relation are not needed for any value below
150  * \f$l_{\text{min}} + 2\f$, so none of the values in the square-roots or
151  * denominators take pathological values for any of the coefficients we require.
152  *
153  * The \f$\beta\f$ constants are filled in member variable `beta_constant`. The
154  * \f$\alpha\f$ values are separately stored as the prefactor for the
155  * \f$\cos(\theta)\f$ term and the constant term in `alpha_prefactor` and
156  * `alpha_constant` member variables.
157  *
158  * In addition, it is efficient to cache recurrence coefficients necessary for
159  * generating the first couple of spin-weighted spherical harmonic functions for
160  * each \f$m\f$ used in the Clenshaw sum.
161  *
162  * The member variable `harmonic_at_l_min_prefactors` holds the prefactors for
163  * directly evaluating the harmonics at \f$s >= m\f$,
164  *
165  * \f{align*}{
166  * {}_s Y_{|s| m} = {}_s C_{m} e^{i m \phi} \sin^a(\theta/2) \cos^b(\theta/2),
167  * \f}
168  *
169  * where \f${}_s C_m\f$ are the cached prefactors that take the values
170  *
171  * \f{align*}{
172  * {}_s C_m = (-1)^{m + \lambda(m)} \sqrt{\frac{(2 |s| + 1) (|s| + k)!}
173  * {4 \pi (|s| + k + a)! (|s| + k + b)!}}
174  * \f}
175  *
176  * and
177  *
178  * \f{align*}{
179  * \lambda(m) =
180  * \left\{
181  * \begin{array}{ll}
182  * 0 &\text{ for } s \ge -m\\
183  * s + m &\text{ for } s < -m
184  * \end{array} \right..
185  * \f}
186  *
187  * The member variable `harmonic_m_recurrence_prefactors` holds the prefactors
188  * necessary to evaluate the lowest harmonics for each \f$m\f$ from the next
189  * lowest-in-magnitude \f$m\f$, allowing most leading harmonics to recursively
190  * derived.
191  *
192  * \f{align*}{
193  * {}_s Y_{|m|, m} = {}_s D_m \sin(\theta / 2) \cos(\theta / 2)
194  * \left\{
195  * \begin{array}{ll}
196  * e^{i \phi} {}_s Y_{|m| - 1, m - 1} &\text{ for } m > 0\\
197  * e^{-i \phi} {}_s Y_{|m| - 1, m + 1} &\text{ for } m < 0
198  * \end{array}\right.,
199  * \f}
200  *
201  * where \f${}_s D_m\f$ are the cached prefactors that take the values
202  *
203  * \f{align*}{
204  * {}_s D_m = -(-1)^{\Delta \lambda(m)} \sqrt{
205  * \frac{(2 |m| + 1)(k + |m| + a + b - 1)(k + |m| + a + b)}
206  * {(2 |m| - 1)(k + |m| + a)(k + |m| + b)}},
207  * \f}
208  * and \f$\Delta \lambda(m)\f$ is the difference in \f$\lambda(m)\f$ between
209  * the harmonic on the left-hand side and right-hand side of the above
210  * recurrence relation, that is \f$\lambda(m) - \lambda(m - 1)\f$ for positive
211  * \f$m\f$ and \f$\lambda(m) - \lambda(m + 1)\f$ for negative \f$m\f$.
212  *
213  * Finally, the member variable
214  * `harmonic_at_l_min_plus_one_recurrence_prefactors` holds the prefactors
215  * necessary to evaluate parts of the recurrence relations from the lowest
216  * \f$l\f$ for a given \f$m\f$ to the next-to-lowest \f$l\f$.
217  *
218  * \f{align*}{
219  * {}_s Y_{l_{\min} + 1, m} = {}_s E_m
220  * \left[(a + 1) + (a + b + 2) \frac{(\cos(\theta) - 1)}{2}\right]
221  * {}_s Y_{l_{\min}, m}.
222  * \f}
223  *
224  * where \f${}_s E_m\f$ are the cached prefactors that take the values
225  *
226  * \f{align*}{
227  * {}_s E_{m} = \sqrt{\frac{2 l_{\min} + 3}{ 2 l_{\min} + 1}}
228  * \sqrt{\frac{(l_{\min} + k + 1)(l_{\min} + k + a + b + 1)}
229  * {(l_{\min} + k + a + 1)(l_{\min} + k + b + 1)}}
230  * \f}
231  */

247  * \brief Perform the Clenshaw recurrence sum, returning by pointer
248  * `interpolated` of interpolating the `goldberg_modes` at the collocation
249  * points passed to the constructor.
250  *
251  * \details The core Clenshaw recurrence is in the\f$l\f$ modes of the
252  * spin-weighted spherical harmonic. For a set of modes \f$a_l^{(a,b)}\f$ at
253  * fixed \f$m\f$, the function value is evaluated by the recurrence:
254  *
255  * \f{align*}{
256  * y^{(a, b)}_{l_\text{max} + 2} &= y^{(a, b)}_{l_\text{max} + 1} = 0 \\
257  * y^{(a, b)}_{l}(\theta) &= \alpha_{l + 1}^{(a, b)} y^{(a, b)}_{l +
258  * 1}(\theta)
259  * + \beta_{l + 2}^{(a,b)} y^{(a, b)}_{l + 2}(\theta) + a_l^{(a, b)} \\
260  * f_m(\theta, \phi) &= \beta_{l_{\text{min}} + 2}
261  * {}_s Y_{l_{\text{min}}, m}(\theta, \phi) y^{(a, b)}_2(\theta) +
262  * {}_s Y_{l_{\text{min}} + 1, m}(\theta, \phi) y^{(a, b)}_1(\theta) +
263  * a_0^{(a, b)} {}_s Y_{l_{\text{min}}, m}(\theta, \phi)
