28 // Compute the coordinate positions of a neighbor element's grid points in the
29 // logical coordinates of the local element.
30 //
31 // However, the results are organized to match the expected input of a
32 // RegularGridInterpolator constructor:
33 // - The coordinates are given as an array of 1D coordinate values.
34 // - For any dimension where the local and neighbor meshes share the same 1D
35 // submesh, the array is empty. (This tells the RegularGridInterpolator that
36 // no interpolation needs to be done in this dimension.)
37 //
38 // Limitations:
39 // - No support for h-refinement. It is ASSERT'ed that the element has only one
40 // neighbor in the specified direction, and that this neighbor is of the same
41 // refinement level as the local element.
42 // - Assumes a uniform rectilinear grid. No support for elements of different
43 // sizes (as could occur if all elements have the same refinement level but
44 // different blocks have different physical size), or curvilinear elements.
45 // This is not checked.
46 template <size_t VolumeDim>
47 std::array<DataVector, VolumeDim> neighbor_grid_points_in_local_logical_coords(
48  const Mesh<VolumeDim>& local_mesh, const Mesh<VolumeDim>& neighbor_mesh,
49  const Element<VolumeDim>& element,
50  const Direction<VolumeDim>& direction_to_neighbor) noexcept;
51
