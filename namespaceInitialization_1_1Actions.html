Detailed Description

Available actions to be used in the initialization.

The action list for initialization must end with the Initialization::Actions::RemoveOptionsAndTerminatePhase action. For example,

using initialization_actions =
tmpl::list<Action0, Action1,
Initialization::Actions::RemoveOptionsAndTerminatePhase>;
