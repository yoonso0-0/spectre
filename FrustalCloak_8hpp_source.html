83  static constexpr Options::String help{
84  "Creates a cubical domain with two equal-sized abutting excised cubes\n"
85  "in the center. This is done by combining ten frusta. The parameter\n"
86  "`UseEquiangularMap` can be used to apply a tangent mapping to the xi\n"
87  "and eta logical coordinates of each frustum, while the parameter\n"
88  "`ProjectionFactor` can be used to apply a projective map to the zeta\n"
89  "logical coordinate of each frustum. Increasing the\n"
90  "`ProjectionFactor` value can give better gridpoint spacings in the\n"
91  "z direction. The user also specifies values for `LengthInnerCube` and\n"
92  "`LengthOuterCube`. This will create a cubical Domain of side"
93  "length `LengthOuterCube` with the center excised. The size of the\n"
94  "excised region is determined by the value set for `LengthInnerCube`.\n"
95  "`OriginPreimage` moves the blocks such that the origin preimage is\n"
96  "mapped to the origin. Note that the abutting excised cubes share a\n"
97  "face in the x-direction. This Domain is primarily for testing the\n"
98  "frustal cloak in the BinaryCompactObject Domain."};
