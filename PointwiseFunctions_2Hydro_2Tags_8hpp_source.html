177 /// The vector \f$J^i\f$ in \f$\dot{M} = -\int J^i s_i d^2S\f$,
178 /// representing the mass flux through a surface with normal \f$s_i\f$.
179 ///
180 /// Note that the integral is understood
181 /// as a flat-space integral: all metric factors are included in \f$J^i\f$.
182 /// In particular, if the integral is done over a Strahlkorper, the
183 /// `StrahlkorperGr::euclidean_area_element` of the Strahlkorper should be used,
184 /// and \f$s_i\f$ is
185 /// the normal one-form to the Strahlkorper normalized with the flat metric,
186 /// \f$s_is_j\delta^{ij}=1\f$.
187 ///
188 /// The formula is
189 /// \f$ J^i = \rho W \sqrt{\gamma}(\alpha v^i-\beta^i)\f$,
190 /// where \f$\rho\f$ is the mass density, \f$W\f$ is the Lorentz factor,
191 /// \f$v^i\f$ is the spatial velocity of the fluid,
192 /// \f$\gamma\f$ is the determinant of the 3-metric \f$\gamma_{ij}\f$,
193 /// \f$\alpha\f$ is the lapse, and \f$\beta^i\f$ is the shift.
194 template <typename DataType, size_t Dim, typename Fr>
195 struct MassFlux : db::SimpleTag {
196  using type = tnsr::I<DataType, Dim, Fr>;
197  static std::string name() noexcept {
198  return Frame::prefix<Fr>() + "MassFlux";
199  }
200 };
