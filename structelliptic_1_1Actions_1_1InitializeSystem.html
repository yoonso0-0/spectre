Detailed Description

template<typename System>
struct elliptic::Actions::InitializeSystem< System >

Initializes the DataBox tags related to the elliptic system.

The system fields are initially set to zero.

Note
Currently the sources are always retrieved from an analytic solution.

Uses:

  • Metavariables:
    • analytic_solution_tag
  • System:
    • fields_tag
    • primal_fields
  • DataBox:
    • Tags::Mesh<Dim>
    • Tags::Coordinates<Dim, Frame::Inertial>

DataBox:

  • Adds:
    • fields_tag
    • db::add_tag_prefix<Tags::FixedSource, fields_tag>
Note
This action relies on the SetupDataBox aggregated initialization mechanism, so Actions::SetupDataBox must be present in the Initialization phase action list prior to this action.
