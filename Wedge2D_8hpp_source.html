27 /*!
28  * \ingroup CoordinateMapsGroup
29  *
30  * \brief Two dimensional map from the logical square to a wedge, which is used
31  * by domains which use disks or annuli.
32  *
33  * \details The coordinate map is constructed by
34  * linearly interpolating between a bulged arc which is circumscribed by a
35  * circular arc of radius `radius_outer` and a bulged arc which is
36  * circumscribed by a circular arc of radius `radius_inner`. These arcs can be
37  * made to be straight or circular, based on the value of `circularity_inner`
38  * or `circularity_outer`, which can take on values between 0 (for straight)
39  * and 1 (for circular). These arcs extend \f$\pi/2\f$ in angle, and can be
40  * oriented along the +/- x or y axis. The choice of using either equiangular
41  * or equidistant coordinates along the arcs is specifiable with
42  * `with_equiangular_map`. The default logical coordinate that points in the
43  * angular direction is \f$\eta\f$. We introduce the auxiliary variable
44  * \f$\mathrm{H}\f$ which is a function of \f$\eta\f$. If we are using
45  * equiangular coordinates, we have:
46  *
47  * \f[\mathrm{H}(\eta) = \textrm{tan}(\eta\pi/4)\f]
48  *
49  * With derivative:
50  *
51  * \f[\mathrm{H}'(\eta) = \frac{\pi}{4}(1+\mathrm{H}^2)\f]
52  *
53  * If we are using equidistant coordinates, we have:
54  *
55  * \f[\mathrm{H}(\eta) = \eta\f]
56  *
57  * with derivative:
58  *
59  * <center>\f$\mathrm{H}'(\eta) = 1\f$</center>
60  *
61  * We also define the variable \f$\rho\f$, given by:
62  *
63  * \f[\rho = \sqrt{1+\mathrm{H}^2}\f]
64  *
65  * In terms of the the circularity \f$c\f$ and the radius \f$R\f$,
66  * the mapping is:
67  *
68  * \f[\vec{x}(\xi,\eta) =
69  * \frac{1}{2}\left\{(1-\xi)\Big[(1-c_{inner})\frac{R_{inner}}{\sqrt 2}
70  * + c_{inner}\frac{R_{inner}}{\rho}\Big] +
71  * (1+\xi)\Big[(1-c_{outer})\frac{R_{outer}}{\sqrt 2} +c_{outer}
72  * \frac{R_{outer}}{\rho}\Big] \right\}\begin{bmatrix}
73  * 1\\
74  * \mathrm{H}\\
75  * \end{bmatrix}\f]
76  *
77  * We will define the variables \f$T(\xi)\f$ and \f$A(\xi)\f$, the trapezoid
78  * and annulus factors: \f[T(\xi) = T_0 + T_1\xi\f] \f[A(\xi) = A_0 +
79  * A_1\xi\f]
80  * Where \f{align*}T_0 &= \frac{1}{2} \big\{ (1-c_{outer})R_{outer} +
81  * (1-c_{inner})R_{inner}\big\}\\
82  * T_1 &= \partial_{\xi} T = \frac{1}{2} \big\{ (1-c_{outer})R_{outer} -
83  * (1-c_{inner})R_{inner}\big\}\\
84  * A_0 &= \frac{1}{2} \big\{ c_{outer}R_{outer} + c_{inner}R_{inner}\big\}\\
85  * A_1 &= \partial_{\xi} A = \frac{1}{2} \big\{ c_{outer}R_{outer} -
86  * c_{inner}R_{inner}\big\}\f}
87  *
88  * The map can then be rewritten as:
89  * \f[\vec{x}(\xi,\eta) = \left\{\frac{T(\xi)}{\sqrt 2} +
90  * \frac{A(\xi)}{\rho}\right\}\begin{bmatrix}
91  * 1\\
92  * \mathrm{H}\\
93  * \end{bmatrix}\f]
94  *
95  *
96  * The Jacobian is: \f[J =
97  * \begin{bmatrix}
98  * \frac{T_1}{\sqrt 2} + \frac{A_1}{\rho} &
99  * \mathrm{H}\mathrm{H}'\frac{A(\xi)}{\rho^3} \\
100  * \mathrm{H}\partial_{\xi}x &
101  * \mathrm{H}\partial_{\eta}x + \mathrm{H}'x\\
102  * \end{bmatrix}
103  * \f]
104  *
105  * The inverse Jacobian is: \f[J^{-1} =
106  * \frac{1}{x}\begin{bmatrix}
107  * \frac{1}{\partial_{\xi}x}\Big\{
108  * \frac{T(\xi)}{\sqrt 2}+\frac{A(\xi)}{\rho^3}
109  * \Big\} & \mathrm{H}\frac{1}{\partial_{\xi}x}\frac{A(\xi)}{\rho^3}\\
110  * -\mathrm{H}\mathrm{H}'^{-1} & \mathrm{H}'^{-1}\\
111  * \end{bmatrix}
112  * \f]
113  *
114  * For a more detailed discussion, see the
115  * documentation for Wedge3D, where the default logical coordinate that points
116  * in the radial direction is \f$\zeta\f$ (In Wedge2D the logical coordinate
117  * that points in the radial direction is \f$\xi\f$), and one may set either
118  * of the two other logical coordinates to zero to obtain an equivalent
119  * Wedge2D map.
120  */
