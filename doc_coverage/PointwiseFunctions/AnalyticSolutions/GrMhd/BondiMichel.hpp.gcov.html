      23             :  * \brief Bondi-Michel accretion \cite Michel1972 with superposed magnetic field
      24             :  * in Schwarzschild spacetime in Cartesian Kerr-Schild coordinates.
      25             :  *
      26             :  * An analytic solution to the 3-D GRMHD system. The user specifies the sonic
      27             :  * radius \f$r_c\f$ and sonic rest mass density \f$\rho_c\f$, which are the
      28             :  * radius and rest mass density at the sonic point, the radius at which the
      29             :  * fluid's Eulerian velocity as seen by a distant observer overtakes the local
      30             :  * sound speed \f$c_{s,c}\f$. With a specified polytropic exponent \f$\gamma\f$,
      31             :  * these quantities can be related to the sound speed at infinity
      32             :  * \f$c_{s,\infty}\f$ using the following relations:
      33             :  *
      34             :  * \f{align*}
      35             :  * c_{s,c}^2 &= \frac{M}{2r_c - 3M} \\
      36             :  * c_{s,\infty}^2 &= \gamma - 1 + (c_{s,c}^2 - \gamma + 1)\sqrt{1 + 3c_{s,c}^2}
      37             :  * \f}
      38             :  *
      39             :  * In the case of the interstellar medium, the sound
      40             :  * speed is \f$\approx 10^{-4}\f$, which results in a sonic radius of
      41             :  * \f$\approx 10^8 M\f$ for \f$\gamma \neq 5/3\f$ \cite RezzollaBook.
      42             :  *
      43             :  * The density is found via root-finding, through the
      44             :  * Bernoulli equation. As one approaches the sonic radius, a second root makes
      45             :  * an appearance and one must take care to bracket the correct root. This is
      46             :  * done by using the upper bound
      47             :  * \f$\frac{\dot{M}}{4\pi}\sqrt{\frac{2}{Mr^3}}\f$.
      48             :  *
      49             :  * Additionally specified by the user are the polytropic exponent \f$\gamma\f$,
      50             :  * and the strength parameter of the magnetic field \f$B_0\f$.
      51             :  * In Cartesian Kerr-Schild coordinates \f$(x, y, z)\f$, where
      52             :  * \f$ r = \sqrt{x^2 + y^2 + z^2}\f$, the superposed magnetic field is
      53             :  * \cite Etienne2010ui
      54             :  *
      55             :  * \f{align*}
      56             :  * B^i(\vec{x},t) = \frac{B_0 M^2}{r^3 \sqrt{\gamma}}x^i
      57             :  *  =\frac{B_0 M^2}{r^3 \sqrt{1 + 2M/r}}x^i.
      58             :  * \f}
      59             :  *
      60             :  * The accretion rate is
      61             :  *
      62             :  * \f{align*}{
      63             :  * \dot{M}=4\pi r^2\rho u^r,
      64             :  * \f}
      65             :  *
      66             :  * and at the sonic radius
      67             :  *
      68             :  * \f{align*}{
      69             :  * \dot{M}_c=\sqrt{8}\pi \sqrt{M}r_c^{3/2}\rho_c.
      70             :  * \f}
      71             :  *
      72             :  * The polytropic constant is given by
      73             :  *
      74             :  * \f{align*}{
      75             :  * K=\frac{1}{\gamma\rho_c^{\gamma-1}}
      76             :  * \left[\frac{M(\gamma-1)}{(2r_c-3M)(\gamma-1)-M}\right].
      77             :  * \f}
      78             :  *
      79             :  * The density as a function of the sound speed is
      80             :  *
      81             :  * \f{align*}{
      82             :  * \rho^{\gamma-1}=\frac{(\gamma-1)c_s^2}{\gamma K(\gamma-1-c_s^2)}.
      83             :  * \f}
      84             :  *
      85             :  * #### Horizon quantities, \f$\gamma\ne5/3\f$
      86             :  * The density at the horizon is given by:
      87             :  *
      88             :  * \f{align*}{
      89             :  *  \rho_h\simeq\frac{1}{16}
      90             :  *  \left(\frac{5-3\gamma}{2}\right)^{(3\gamma-5)/[2(\gamma-1)]}
      91             :  *  \frac{\rho_\infty}{c_{s,\infty}^3}.
      92             :  * \f}
      93             :  *
      94             :  * Using the Lorentz invariance of \f$b^2\f$ we evaluate:
      95             :  *
      96             :  * \f{align*}{
      97             :  * b^2=\frac{B^2}{W^2}+(B^i v_i)^2=
      98             :  *  B^r B^r(1-\gamma_{rr}v^r v^r)+B^r B^r v^r v^r
      99             :  * =B^r B^r = \frac{B_0^2 M^4}{r^4},
     100             :  * \f}
     101             :  *
     102             :  * where \f$r\f$ is the Cartesian Kerr-Schild radius, which is equal to the
     103             :  * areal radius for a non-spinning black hole. At the horizon we get
     104             :  *
     105             :  * \f{align*}{
     106             :  * b^2_h=\frac{B^2_0}{16}.
     107             :  * \f}
     108             :  *
     109             :  * Finally, we get
     110             :  *
     111             :  * \f{align*}{
     112             :  * B_0 = 4 \sqrt{b^2_h} = 4\sqrt{\rho_h} \sqrt{\frac{b^2_h}{\rho_h}},
     113             :  * \f}
     114             :  *
     115             :  * where the last equality is useful for comparison to papers that give
     116             :  * \f$b^2_h/\rho_h\f$.
     117             :  *
     118             :  * To help with comparing to other codes the following script can be used to
     119             :  * compute \f$b^2_h/\rho_h\f$:
     120             :  *
     121             :  * \code{.py}
     122             :  * #!/bin/env python
     123             :  *
     124             :  * import numpy as np
     125             :  *
     126             :  * # Input parameters
     127             :  * B_0 = 18
     128             :  * r_c = 8
     129             :  * rho_c = 1 / 16
     130             :  * gamma = 4 / 3
     131             :  * mass = 1
     132             :  *
     133             :  * K = 1 / (gamma * rho_c**(gamma - 1)) * ((gamma - 1) * mass) / (
     134             :  *     (2 * r_c - 3 * mass) * (gamma - 1) - mass)
     135             :  * c_s_c = mass / (2 * r_c - 3 * mass)
     136             :  * c_inf = gamma - 1 + (c_s_c - gamma + 1) * np.sqrt(1. + 3. * c_s_c)
     137             :  * rho_inf = ((gamma - 1) * c_inf / (gamma * K *
     138             :  *                                   (gamma - 1 - c_inf)))**(1. / (gamma - 1.))
     139             :  * rho_h = 1. / 16. * (2.5 - 1.5 * gamma)**(
     140             :  *     (3 * gamma - 5) / (2 * (gamma - 1))) * rho_inf / (c_inf**1.5)
     141             :  *
     142             :  * print("B_0", B_0)
     143             :  * print("r_c: ", r_c)
     144             :  * print("rho_c", rho_c)
     145             :  * print("b_h^2/rho_h: ", B_0**2 / (16. * rho_h))
     146             :  * print("gamma: ", gamma)
     147             :  * \endcode
     148             :  *
     149             :  * #### Horizon quantities, \f$\gamma=5/3\f$
     150             :  * The density at the horizon is given by:
     151             :  *
     152             :  * \f{align*}{
     153             :  * \rho_h\simeq \frac{1}{16}\frac{\rho_\infty}{u_h c_{s,\infty}^3},
     154             :  * \f}
     155             :  *
     156             :  * which gives \cite RezzollaBook
     157             :  *
     158             :  * \f{align*}{
     159             :  * \rho_h\simeq 0.08\frac{\rho_\infty}{c_{s,\infty}^3}.
     160             :  * \f}
     161             :  *
     162             :  * The magnetic field \f$b^2\f$ is the same as the \f$\gamma\ne5/3\f$.
     163             :  */
