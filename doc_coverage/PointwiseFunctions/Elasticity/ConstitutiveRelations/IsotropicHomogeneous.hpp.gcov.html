      26             :  * \details For an isotropic and homogeneous material the linear constitutive
      27             :  * relation \f$T^{ij}=-Y^{ijkl}S_{kl}\f$ reduces to
      28             :  *
      29             :  * \f[
      30             :  * Y^{ijkl} = \lambda \delta^{ij}\delta^{kl} + \mu
      31             :  * \left(\delta^{ik}\delta^{jl} + \delta^{il}\delta^{jk}\right) \\
      32             :  * \implies \quad T^{ij} = -\lambda \mathrm{Tr}(S) \delta^{ij} - 2\mu S^{ij}
      33             :  * \f]
      34             :  *

      36             :  * _rigidity_) \f$\mu\f$. In the parametrization chosen in this implementation
      37             :  * we use the _bulk modulus_ (or _incompressibility_)
      38             :  *
      39             :  * \f[
      40             :  * K=\lambda + \frac{2}{3}\mu
      41             :  * \f]

      43             :  * instead of the Lamé parameter. In this parametrization the
      44             :  * stress-strain relation

      47             :  * T^{ij} = -K \mathrm{Tr}(S) \delta^{ij} - 2\mu\left(S^{ij} -
