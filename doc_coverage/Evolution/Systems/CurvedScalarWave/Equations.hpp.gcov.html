     146             :  * \brief Computes the upwind flux for scalar-waves in curved spacetime.
     147             :  *
     148             :  * \details The upwind flux is given by \cite Teukolsky2015ega :
     149             :  *
     150             :  * \f[
     151             :  * G = S\left(\Lambda^+ S^{-1} U^{\rm int}
     152             :  *           + \Lambda^- S^{-1} U^{\rm ext}\right)
     153             :  * = S \left(\Lambda^+ \hat{U}^{\rm int}
     154             :  *           + \Lambda^- \hat{U}^{\rm ext}\right),
     155             :  * \f]
     156             :  *
     157             :  * where
     158             :  *
     159             :  *  - \f$G\f$ is the numerical upwind flux dotted with the interface normal;
     160             :  *  - \f$U\f$ is a vector of all evolved variables;
     161             :  *  - \f$S\f$ is a matrix whose columns are the eigenvectors of the
     162             :  *       characteristic matrix for the evolution system. It maps the
     163             :  *       evolved variables to characteristic variables \f$\hat{U}\f$, s.t.
     164             :  *       \f$\hat{U} := S^{-1}\cdot U\f$; and
     165             :  *  - \f$\Lambda^\pm\f$ are diagonal matrices containing
     166             :  *       positive / negative eigenvalues of the same matrix as its elements.
     167             :  *
     168             :  * The superscripts \f${\rm int}\f$ and \f${\rm ext}\f$ on \f$U\f$ indicate
     169             :  * that the corresponding set of variables at the element interface have been
     170             :  * taken from the _interior_ or _exterior_ of the element. Exterior of the
     171             :  * element is naturally the interior of its neighboring element. Therefore,
     172             :  * \f$\hat{U}^{\rm int}:= S^{-1}U^{\rm int}\f$ are the characteristic variables
     173             :  * at the element interface computed using evolved variables from the interior
     174             :  * of the element, and \f$\hat{U}^{\rm ext}= S^{-1}U^{\rm ext}\f$ are the
     175             :  * same computed from evolved variables taken from the element exterior, i.e.
     176             :  * the neighboring element.
     177             :  * The sign of characteristic speeds indicates the direction of propagation of
     178             :  * the corresponding characteristic field with respect to the interface normal
     179             :  * that the field has been computed along (with negative speeds indicating
     180             :  * incoming characteristics, and with positive speeds indicating outgoing
     181             :  * characteristics). Therefore, \f$\Lambda^+\f$ contains characterstic speeds
     182             :  * for variables that are outgoing from the element at the interface, and
     183             :  * \f$\Lambda^-\f$ contains characteristic speeds for variables that are
     184             :  * incoming to the element at the interface. An ambiguity naturally arises
     185             :  * as to which set of evolved variables (''interior'' or ''exterior'') to use
     186             :  * when computing these speeds. We compute both and use their average, i.e.
     187             :  * \f$\lambda^{\rm avg} = (\lambda^{\rm int} + \lambda^{\rm ext})/2\f$, to
     188             :  * populate \f$\Lambda^\pm\f$.
     189             :  *
     190             :  *
     191             :  * This function computes the upwind flux as follows:
     192             :  *  -# Computes internal and external characteristic variables and speeds using
     193             :  * evolved variables from
     194             :  *     both the element interior and exterior.
     195             :  *  -# Computes the average characteristic speeds and constructs
     196             :  *      \f$\Lambda^{\pm}\f$.
     197             :  *  -# Computes the upwind flux as a weigted sum of external and internal
     198             :  *     characteristic variables
     199             :  *
     200             :  * \f[
     201             :  * G = S\left(w_{\rm ext} \hat{U}^{\rm ext}
     202             :  *       + w_{\rm int} \hat{U}^{\rm int}\right),
     203             :  * \f]
     204             :  *
     205             :  * with weights \f$w_{\rm ext} = \Theta(-\Lambda)\cdot\Lambda\f$, and
     206             :  * \f$w_{\rm int} = \Theta(\Lambda)\cdot\Lambda\f$, where \f$\Theta\f$ is the
     207             :  * step function centered at zero, \f$\Lambda = \Lambda^+ + \Lambda^-\f$, and
     208             :  * the dot operator \f$(\cdot)\f$ indicates an element-wise product.
     209             :  *
     210             :  * \warning With the averaging of characteristic speeds, this flux does not
     211             :  * satisfy the generalized Rankine-Hugoniot conditions
     212             :  * \f${G}^{\rm int}(\hat{U}^{\rm int}, \hat{U}^{\rm ext})
     213             :  * = - {G}^{\rm ext}(\hat{U}^{\rm ext}, \hat{U}^{\rm int})\f$,
     214             :  * which enforces that the flux leaving the element is equal to the flux
     215             :  * entering the neighboring element, and vice versa. This condition is
     216             :  * important for a well-balanced scheme, and so please use this flux with
     217             :  * caution.
     218             :  */
