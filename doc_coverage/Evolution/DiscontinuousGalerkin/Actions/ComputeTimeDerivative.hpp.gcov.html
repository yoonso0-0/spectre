      69             :  *
      70             :  * The general first-order hyperbolic evolution equation solved for conservative
      71             :  * systems is:
      72             :  *
      73             :  * \f{align*}{
      74             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      75             :  *  + \partial_{i}
      76             :  *   \left(F^i_\alpha - v^i_g u_\alpha\right)
      77             :  *   = S_\alpha-u_\alpha\partial_i v^i_g,
      78             :  * \f}
      79             :  *
      80             :  * where \f$F^i_{\alpha}\f$ are the fluxes when the mesh isn't moving,
      81             :  * \f$v^i_g\f$ is the velocity of the mesh, \f$u_{\alpha}\f$ are the evolved
      82             :  * variables, \f$S_{\alpha}\f$ are the source terms, and \f$\hat{t}\f$ is the
      83             :  * time in the logical frame. For evolution equations that do not have any
      84             :  * fluxes and only nonconservative products we evolve:
      85             :  *
      86             :  * \f{align*}{
      87             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      88             :  *   +\left(B^i_{\alpha\beta}-v^i_g \delta_{\alpha\beta}
      89             :  *   \right)\partial_{i}u_\beta = S_\alpha.
      90             :  * \f}
      91             :  *
      92             :  * Finally, for equations with both conservative terms and nonconservative
      93             :  * products we use:
      94             :  *
      95             :  * \f{align*}{
      96             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      97             :  *   + \partial_{i}
      98             :  *   \left(F^i_\alpha - v^i_g u_\alpha\right)
      99             :  *   +B^i_{\alpha\beta}\partial_{i}u_\beta
     100             :  *   = S_\alpha-u_\alpha\partial_i v^i_g,
     101             :  * \f}
