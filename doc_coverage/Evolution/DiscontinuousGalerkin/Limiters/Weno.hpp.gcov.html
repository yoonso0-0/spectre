      66             : /// The limiter uses the minmod-based TVB troubled-cell indicator (TCI) of
      67             : /// \cite Cockburn1999 to identify elements that need limiting. The simple
      68             : /// WENO implementation follows the paper: it checks the TCI independently
      69             : /// to each tensor component, so that only certain tensor components may be
      70             : /// limited. The HWENO implementation checks the TCI for all tensor components,
      71             : /// and if any single component is troubled, then all components are limited.

      73             : /// Newtonian Euler fluid system, uses a more sophisticated TCI that is adapted

      75             : /// because it is easily applied to a general set of tensors.
      76             : ///
      77             : /// For each tensor component to limit, the new solution is obtained by WENO
      78             : /// reconstruction --- a linear combination of the local DG solution and a
      79             : /// "modified" solution from each neighbor element. For the simple WENO limiter,
      80             : /// the modified solution is obtained by simply extrapolating the neighbor
      81             : /// solution onto the troubled element. For the HWENO limiter, the modified
      82             : /// solution is obtained by a least-squares fit to the solution across multiple
      83             : /// neighboring elements.
