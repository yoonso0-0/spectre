      27             :  * \brief Three dimensional map from the cube to a wedge.
      28             :  * \image html Shell.png "A shell can be constructed out of six wedges."
      29             :  *
      30             :  * \details The mapping that goes from a reference cube to a three-dimensional
      31             :  *  wedge centered on a coordinate axis covering a volume between an inner
      32             :  *  surface and outer surface. Each surface can be given a curvature
      33             :  *  between flat (a sphericity of 0) or spherical (a sphericity of 1).
      34             :  *
      35             :  *  The first two logical coordinates correspond to the two angular coordinates,
      36             :  *  and the third to the radial coordinate.
      37             :  *
      38             :  *  The Wedge3D map is constructed by linearly interpolating between a bulged
      39             :  *  face of radius `radius_of_inner_surface` to a bulged face of
      40             :  *  radius `radius_of_outer_surface`, where the radius of each bulged face
      41             :  *  is defined to be the radius of the sphere circumscribing the bulge.
      42             :  *
      43             :  *  We make a choice here as to whether we wish to use the logical coordinates
      44             :  *  parameterizing these surface as they are, in which case we have the
      45             :  *  equidistant choice of coordinates, or whether to apply a tangent map to them
      46             :  *  which leads us to the equiangular choice of coordinates. In terms of the
      47             :  *  logical coordinates, the equiangular coordinates are:
      48             :  *
      49             :  *  \f[\textrm{equiangular xi} : \Xi(\xi) = \textrm{tan}(\xi\pi/4)\f]
      50             :  *
      51             :  *  \f[\textrm{equiangular eta}  : \mathrm{H}(\eta) = \textrm{tan}(\eta\pi/4)\f]
      52             :  *
      53             :  *  With derivatives:
      54             :  *
      55             :  *  \f[\Xi'(\xi) = \frac{\pi}{4}(1+\Xi^2)\f]
      56             :  *
      57             :  *  \f[\mathrm{H}'(\eta) = \frac{\pi}{4}(1+\mathrm{H}^2)\f]
      58             :  *
      59             :  *  The equidistant coordinates are:
      60             :  *
      61             :  *  \f[ \textrm{equidistant xi}  : \Xi = \xi\f]
      62             :  *
      63             :  *  \f[ \textrm{equidistant eta}  : \mathrm{H} = \eta\f]
      64             :  *
      65             :  *  with derivatives:
      66             :  *
      67             :  *  <center>\f$\Xi'(\xi) = 1\f$, and \f$\mathrm{H}'(\eta) = 1\f$</center>
      68             :  *
      69             :  *  We also define the variable \f$\rho\f$, given by:
      70             :  *
      71             :  *  \f[\textrm{rho} : \rho = \sqrt{1+\Xi^2+\mathrm{H}^2}\f]
      72             :  *
      73             :  *  ### The Spherical Face Map
      74             :  *  The surface map for the spherical face of radius \f$R\f$ lying in the
      75             :  * \f$+z\f$
      76             :  *  direction in either choice of coordinates is then given by:
      77             :  *
      78             :  *  \f[\vec{\sigma}_{spherical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      79             :  *  Where
      80             :  *  \f[
      81             :  *  \vec{x}(\xi,\eta) =
      82             :  *  \begin{bmatrix}
      83             :  *  x(\xi,\eta)\\
      84             :  *  y(\xi,\eta)\\
      85             :  *  z(\xi,\eta)\\
      86             :  *  \end{bmatrix}  = \frac{R}{\rho}
      87             :  *  \begin{bmatrix}
      88             :  *  \Xi\\
      89             :  *  \mathrm{H}\\
      90             :  *  1\\
      91             :  *  \end{bmatrix}\f]
      92             :  *
      93             :  *  ### The Bulged Face Map
      94             :  *  The bulged surface is itself constructed by linearly interpolating between
      95             :  *  a cubical face and a spherical face. The surface map for the cubical face
      96             :  *  of side length \f$2L\f$ lying in the \f$+z\f$ direction is given by:
      97             :  *
      98             :  *  \f[\vec{\sigma}_{cubical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      99             :  *  Where
     100             :  *  \f[
     101             :  *  \vec{x}(\xi,\eta) =
     102             :  *  \begin{bmatrix}
     103             :  *  x(\xi,\eta)\\
     104             :  *  y(\xi,\eta)\\
     105             :  *  L\\
     106             :  *  \end{bmatrix}  = L
     107             :  *  \begin{bmatrix}
     108             :  *  \Xi\\
     109             :  *  \mathrm{H}\\
     110             :  *  1\\
     111             :  *  \end{bmatrix}\f]
     112             :  *
     113             :  *  To construct the bulged map we interpolate between this cubical face map
     114             :  *  and a spherical face map of radius \f$R\f$, with the
     115             :  *  interpolation parameter being \f$s\f$. The surface map for the bulged face
     116             :  *  lying in the \f$+z\f$ direction is then given by:
     117             :  *
     118             :  *  \f[\vec{\sigma}_{bulged}(\xi,\eta) = {(1-s)L + \frac{sR}{\rho}}
     119             :  *  \begin{bmatrix}
     120             :  *  \Xi\\
     121             :  *  \mathrm{H}\\
     122             :  *  1\\
     123             :  *  \end{bmatrix}\f]
     124             :  *
     125             :  *  We constrain L by demanding that the spherical face circumscribe the cube.
     126             :  *  With this condition, we have \f$L = R/\sqrt3\f$.
     127             :  *  \note This differs from the choice in SpEC where it is demanded that the
     128             :  *  surfaces touch at the center, which leads to \f$L = R\f$.
     129             :  *
     130             :  *  ### The Full Volume Map
     131             :  *  The final map for the wedge which lies along the \f$+z\f$ is obtained by
     132             :  *  interpolating between the two surfaces with the
     133             :  *  interpolation parameter being the logical coordinate \f$\zeta\f$. This
     134             :  *  results in:
     135             :  *
     136             :  *  \f[\vec{x}(\xi,\eta,\zeta) =
     137             :  *  \frac{1}{2}\left\{(1-\zeta)\Big[(1-s_{inner})\frac{R_{inner}}{\sqrt 3}
     138             :  *   + s_{inner}\frac{R_{inner}}{\rho}\Big] +
     139             :  *  (1+\zeta)\Big[(1-s_{outer})\frac{R_{outer}}{\sqrt 3} +s_{outer}
     140             :  *  \frac{R_{outer}}{\rho}\Big] \right\}\begin{bmatrix}
     141             :  *  \Xi\\
     142             :  *  \mathrm{H}\\
     143             :  *  1\\
     144             :  *  \end{bmatrix}\f]
     145             :  *
     146             :  *  We will define the variables \f$F(\zeta)\f$ and \f$S(\zeta)\f$, the frustum
     147             :  * and sphere factors: \f[F(\zeta) = F_0 + F_1\zeta\f] \f[S(\zeta) = S_0 +
     148             :  * S_1\zeta\f]
     149             :  *  Where \f{align*}F_0 &= \frac{1}{2} \big\{ (1-s_{outer})R_{outer} +
     150             :  * (1-s_{inner})R_{inner}\big\}\\
     151             :  *  F_1 &= \partial_{\zeta} F = \frac{1}{2} \big\{ (1-s_{outer})R_{outer} -
     152             :  * (1-s_{inner})R_{inner}\big\}\\
     153             :  *  S_0 &= \frac{1}{2} \big\{ s_{outer}R_{outer} + s_{inner}R_{inner}\big\}\\
     154             :  *  S_1 &= \partial_{\zeta} S = \frac{1}{2} \big\{ s_{outer}R_{outer} -
     155             :  * s_{inner}R_{inner}\big\}\f}
     156             :  *
     157             :  *  The map can then be rewritten as:
     158             :  * \f[\vec{x}(\xi,\eta,\zeta) = \left\{\frac{F(\zeta)}{\sqrt 3} +
     159             :  * \frac{S(\zeta)}{\rho}\right\}\begin{bmatrix}
     160             :  *  \Xi\\
     161             :  *  \mathrm{H}\\
     162             :  *  1\\
     163             :  *  \end{bmatrix}\f]
     164             :  *
     165             :  *  We provide some common derivatives:
     166             :  *  \f[\partial_{\xi}z = \frac{-S(\zeta)\Xi\Xi'}{\rho^3}\f]
     167             :  *  \f[\partial_{\eta}z = \frac{-S(\zeta)\mathrm{H}\mathrm{H}'}{\rho^3}\f]
     168             :  * \f[\partial_{\zeta}z = \frac{F'}{\sqrt 3} + \frac{S'}{\rho}\f]
     169             :  *  The Jacobian then is: \f[J =
     170             :  *  \begin{bmatrix}
     171             :  *  \Xi'z + \Xi\partial_{\xi}z & \Xi\partial_{\eta}z & \Xi\partial_{\zeta}z \\
     172             :  *  \mathrm{H}\partial_{\xi}z & \mathrm{H}'z +
     173             :  *  \mathrm{H}\partial_{\eta}z & \mathrm{H}\partial_{\zeta}z\\
     174             :  *   \partial_{\xi}z&\partial_{\eta}z &\partial_{\zeta}z \\
     175             :  *  \end{bmatrix}
     176             :  *  \f]
     177             :  *
     178             :  *  A common factor that shows up in the inverse jacobian is:
     179             :  *  \f[ T:= \frac{S(\zeta)}{(\partial_{\zeta}z)\rho^3}\f]
     180             :  *
     181             :  *  The inverse Jacobian then is: \f[J^{-1} =
     182             :  *  \frac{1}{z}\begin{bmatrix}
     183             :  *  \Xi'^{-1} & 0 & -\Xi\Xi'^{-1}\\
     184             :  *  0 & \mathrm{H}'^{-1} & -\mathrm{H}\mathrm{H}'^{-1}\\
     185             :  *  T\Xi &
     186             :  *  T\mathrm{H} &
     187             :  *  T + F(\partial_{\zeta}z)^{-1}/\sqrt 3\\
     188             :  *  \end{bmatrix}
     189             :  *  \f]
     190             :  *
     191             :  *  ### Changing the radial distribution of the gridpoints
     192             :  *  By default, Wedge3D linearly distributes its gridpoints in the radial
     193             :  *  direction. An exponential distribution of gridpoints can be obtained by
     194             :  *  linearly interpolating in the logarithm of the radius, in order to obtain
     195             :  *  a relatively higher resolution at smaller radii. Since this is a radial
     196             :  *  rescaling of Wedge3D, this option is only supported for fully spherical
     197             :  *  wedges with `sphericity_inner` = `sphericity_outer` = 1.
     198             :  *
     199             :  *  The linear interpolation done is:
     200             :  *  \f[
     201             :  *  \log r = \frac{1-\zeta}{2}\log R_{inner} +
     202             :  *  \frac{1+\zeta}{2}\log R_{outer}
     203             :  *  \f]
     204             :  *
     205             :  *  The map then is:
     206             :  *  \f[\vec{x}(\xi,\eta,\zeta) =
     207             :  *  \frac{\sqrt{R_{inner}^{1-\zeta}R_{outer}^{1+\zeta}}}{\rho}\begin{bmatrix}
     208             :  *  \Xi\\
     209             :  *  \mathrm{H}\\
     210             :  *  1\\
     211             :  *  \end{bmatrix}\f]
     212             :  *
     213             :  *  The jacobian simplifies similarly.
     214             :  *
     215             :  */
