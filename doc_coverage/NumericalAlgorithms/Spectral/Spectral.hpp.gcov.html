     271             : /*!
     272             :  * \brief %Matrix used to perform an indefinite integral of a function over the
     273             :  * logical grid. The left boundary condition is such that the integral is 0 at
     274             :  * \f$\xi=-1\f$
     275             :  *
     276             :  * Currently only Legendre and Chebyshev polynomials are implemented, but we
     277             :  * provide a derivation for how to compute the indefinite integration matrix for
     278             :  * general Jacobi polynomials.
     279             :  *
     280             :  * #### Legendre Polynomials
     281             :  * The Legendre polynomials have the identity:
     282             :  *
     283             :  * \f{align*}{
     284             :  * P_n(x) = \frac{1}{2n+1}\frac{d}{dx}\left(P_{n+1}(x) - P_{n-1}(x)\right)
     285             :  * \f}
     286             :  *
     287             :  * The goal is to evaluate the integral of a function \f$u\f$ expanded in terms
     288             :  * of Legendre polynomials as
     289             :  *
     290             :  * \f{align*}{
     291             :  * u(x) = \sum_{i=0}^{N} c_i P_i(x)
     292             :  * \f}
     293             :  *
     294             :  * We similarly expand the indefinite integral of \f$u\f$ as
     295             :  *
     296             :  * \f{align*}{
     297             :  * \left.\int u(y) dy\right\rvert_{x}=&\sum_{i=0}^N \tilde{c}_i P_i(x) \\
     298             :  *   =&\left.\int\sum_{i=1}^{N}\frac{c_i}{2i+1}
     299             :  *      \left(P_{i+1}(y)-P_{i-1}(y)\right)dy\right\rvert_{x}
     300             :  *      + \tilde{c}_0 P_0(x) \\
     301             :  *   =&\sum_{i=1}^{N}\left(\frac{c_{i-1}}{2i-1} - \frac{c_{i+1}}{2i+3}\right)
     302             :  *     P_i(x) + \tilde{c}_0 P_0(x)
     303             :  * \f}
     304             :  *
     305             :  * Thus we get that for \f$i>0\f$
     306             :  *
     307             :  * \f{align*}{
     308             :  * \tilde{c}_i=\frac{c_{i-1}}{2i-1}-\frac{c_{i+1}}{2i+3}
     309             :  * \f}
     310             :  *
     311             :  * and \f$\tilde{c}_0\f$ is a constant of integration, which we choose such that
     312             :  * the integral is 0 at the left boundary of the domain (\f$x=-1\f$). The
     313             :  * condition for this is:
     314             :  *
     315             :  * \f{align*}{
     316             :  *   \tilde{c}_0=\sum_{i=1}^{N}(-1)^{i+1}\tilde{c}_i
     317             :  * \f}
     318             :  *
     319             :  * The matrix returned by this function is the product of the tridiagonal matrix
     320             :  * for the \f$\tilde{c}_i\f$ and the matrix for the boundary condition.
     321             :  *
     322             :  * #### Chebyshev Polynomials
     323             :  *
     324             :  * A similar derivation leads to the relations:
     325             :  *
     326             :  * \f{align*}{
     327             :  *  \tilde{c}_i=&\frac{c_{i-1}-c_{i+1}}{2i},&\mathrm{if}\;i>1 \\
     328             :  *  \tilde{c}_1=&c_0 - \frac{c_2}{2},&\mathrm{if}\;i=1 \\
     329             :  * \f}
     330             :  *
     331             :  * We again have:
     332             :  *
     333             :  * \f{align*}{
     334             :  * \tilde{c}_0=\sum_{i=1}^N(-1)^{i+1}\tilde{c}_i
     335             :  * \f}
     336             :  *
     337             :  * These are then used to define the indefinite integration matrix.
     338             :  *
     339             :  * #### Jacobi Polynomials
     340             :  *
     341             :  * For general Jacobi polynomials \f$P^{(\alpha,\beta)}_n(x)\f$ given by
     342             :  *
     343             :  * \f{align*}{
     344             :  *  (1-x)^\alpha(1+x)^\beta P^{(\alpha,\beta)}_n(x)=\frac{(-1)^n}{2^n n!}
     345             :  *  \frac{d^n}{dx^n}\left[(1-x)^{\alpha+n}(1+x)^{\beta+n}\right]
     346             :  * \f}
     347             :  *
     348             :  * we have that
     349             :  *
     350             :  * \f{align*}{
     351             :  * P^{(\alpha,\beta)}_n(x)=\frac{d}{dx}\left(
     352             :  * b^{(\alpha,\beta)}_{n-1,n}P^{(\alpha,\beta)}_{n-1}(x) +
     353             :  * b^{(\alpha,\beta)}_{n,n}P^{(\alpha,\beta)}_n(x) +
     354             :  * b^{(\alpha,\beta)}_{n+1,n}P^{(\alpha,\beta)}_{n+1}(x)
     355             :  * \right)
     356             :  * \f}
     357             :  *
     358             :  * where
     359             :  *
     360             :  * \f{align*}{
     361             :  * b^{(\alpha,\beta)}_{n-1,n}=&-\frac{1}{n+\alpha+\beta}
     362             :  *                            a^{(\alpha,\beta)}_{n-1,n} \\
     363             :  * b^{(\alpha,\beta)}_{n,n}=&-\frac{2}{\alpha+\beta}
     364             :  *                          a^{(\alpha,\beta)}_{n,n} \\
     365             :  * b^{(\alpha,\beta)}_{n+1,n}=&\frac{1}{n+1}
     366             :  *                            a^{(\alpha,\beta)}_{n+1,n} \\
