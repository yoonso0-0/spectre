Detailed Description

A struct that stores the charm++ types relevant for a particular array component.

Details

The type traits are:

  • cproxy: the charm++ proxy. See https://charm.readthedocs.io/en/latest/faq/manual.html#what-is-a-proxy
  • cbase: the charm++ base class. See https://charm.readthedocs.io/en/latest/charm++/manual.html#chare-objects
  • algorithm_type: the chare type (AlgorithmArray) for the array component.
  • ckindex: A charm++ chare index object. Useful for obtaining entry method indices that are needed for creating callbacks. See https://charm.readthedocs.io/en/latest/charm++/manual.html#creating-a-ckcallback-object
