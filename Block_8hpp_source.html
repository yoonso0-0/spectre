29 /// \ingroup ComputationalDomainGroup
30 /// A Block<VolumeDim> is a region of a VolumeDim-dimensional computational
31 /// domain that defines the root node of a tree which is used to construct the
32 /// Elements that cover a region of the computational domain.
33 ///
34 /// Each codimension 1 boundary of a Block<VolumeDim> is either an external
35 /// boundary or identical to a boundary of one other Block.
36 ///
37 /// A Block has logical coordinates that go from -1 to +1 in each
38 /// dimension. The global coordinates are obtained from the logical
39 /// coordinates from the Coordinatemap: CoordinateMap::operator() takes
40 /// Points in the Logical Frame (i.e., logical coordinates) and
41 /// returns Points in the Inertial Frame (i.e., global coordinates).
42 template <size_t VolumeDim>
43 class Block {
44  public:
45  /// \param stationary_map the CoordinateMap.
46  /// \param id a unique ID.

48  /// boundary with this Block.
