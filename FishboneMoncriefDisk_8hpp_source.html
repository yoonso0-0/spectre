31 /*!
32  * \brief Fluid disk orbiting a Kerr black hole
33  *
34  * The Fishbone-Moncrief solution to the 3D relativistic Euler system
35  * \cite Fishbone1976apj, representing the isentropic flow of a thick fluid disk
36  * orbiting a Kerr black hole. In Boyer-Lindquist coordinates
37  * \f$(t, r, \theta, \phi)\f$, the flow is assumed to be purely toroidal,
38  *
39  * \f{align*}
40  * u^\mu = (u^t, 0, 0, u^\phi),
41  * \f}
42  *
43  * where \f$u^\mu\f$ is the 4-velocity. Then, all the fluid quantities are
44  * assumed to share the same symmetries as those of the background spacetime,
45  * namely they are stationary (independent of \f$t\f$), and axially symmetric
46  * (independent of \f$\phi\f$). Note that \f$r\f$ and \f$\theta\f$ can also be
47  * interpreted as Kerr (a.k.a. "spherical" Kerr-Schild) coordinates
48  * (see gr::KerrSchildCoords), and that the symmetries of the equilibrium ensure
49  * that \f$u^\mu\f$ as defined above is also the 4-velocity in Kerr coordinates.
50  *
51  * Self-gravity is neglected, so that the fluid
52  * variables are determined as functions of the metric. Following the treatment
53  * by Kozlowski et al. \cite Kozlowski1978aa (but using signature +2)
54  * the solution is expressed in terms of the quantities
55  *
56  * \f{align*}
57  * \Omega &= \dfrac{u^\phi}{u^t},\\
58  * W &= W_\text{in} - \int_{p_\text{in}}^p\frac{dp}{e + p},
59  * \f}
60  *
61  * where \f$\Omega\f$ is the angular velocity, \f$p\f$ is the fluid pressure,
62  * \f$e\f$ is the energy density, and \f$W\f$ is an auxiliary quantity
63  * interpreted in the Newtonian limit as the total (gravitational + centrifugal)
64  * potential. \f$W_\text{in}\f$ and \f$p_\text{in}\f$ are the potential and
65  * the pressure at the radius of the inner edge, i.e. the closest edge to the
66  * black hole. Here we assume \f$p_\text{in} = 0.\f$ The solution to the Euler
67  * equation is then
68  *
69  * \f{align*}
70  * (u^t)^2 &= \frac{A}{2\Delta\Sigma}\left(1 +
71  * \sqrt{1 + \frac{4l^2\Delta \Sigma^2}{A^2\sin^2\theta}}\right)\\
72  * \Omega &= \frac{\Sigma}{A (u^t)^2}\frac{l}{\sin^2\theta} + \frac{2Mra}{A}\\
73  * u^\phi &= \Omega u^t\\
74  * W &= l\Omega - \ln u^t,
75  * \f}
76  *
77  * where
78  *
79  * \f{align*}
80  * \Sigma = r^2 + a^2\cos^2\theta\qquad
81  * \Delta = r^2 - 2Mr + a^2\qquad
82  * A = (r^2 + a^2)^2 - \Delta a^2 \sin^2\theta
83  * \f}
84  *
85  * and \f$l = u_\phi u^t\f$ is the so-called angular momentum per unit
86  * intertial mass, which is a parameter defining an
87  * individual disk. In deriving the solution, an integration constant has been
88  * chosen so that \f$ W\longrightarrow 0\f$ as \f$r\longrightarrow \infty\f$,
89  * in accordance with the Newtonian limit. Note that, from its definition,
90  * equipotential contours coincide with isobaric contours. Physically, the
91  * matter can fill each of the closed surfaces \f$W = \text{const}\f$, giving
92  * rise to an orbiting thick disk. For \f$W > 0\f$, all equipotentials are open,
93  * whereas for \f$W < 0\f$, some of them will be closed. Should a disk exist,
94  * the pressure reaches a maximum value on the equator at a coordinate radius
95  * \f$r_\text{max}\f$ that is related to the angular momentum per unit inertial
96  * mass via
97  *
98  * \f{align*}
99  * l = \dfrac{M^{1/2}(r_\text{max}^{3/2} + aM^{1/2})(a^2 - 2aM^{1/2}
100  * r_\text{max}^{1/2} + r_\text{max}^2)}{2aM^{1/2}r_\text{max}^{3/2} +
101  * (r_\text{max} - 3M)r_\text{max}^2}.
102  * \f}
103  *
104  * Once \f$W\f$ is determined, an equation of state is required in order to
105  * obtain the thermodynamic variables. If the flow is isentropic, the specific
106  * enthalpy can readily be obtained from the first and second laws of
107  * thermodynamics: one has
108  *
109  * \f{align*}
110  * \frac{dp}{e + p} = \frac{dh}{h}
111  * \f}
112  *
113  * so that
114  *
115  * \f{align*}
116  * h = h_\text{in}\exp(W_\text{in} - W),
117  * \f}
118  *
119  * and the pressure can be obtained from a thermodynamic relation of the form
120  * \f$h = h(p)\f$. Here we assume a polytropic relation
121  *
122  * \f{align*}
123  * p = K\rho^\gamma.
124  * \f}
125  *
126  * Once all the variables are known in Boyer-Lindquist (or Kerr) coordinates, it
127  * is straightforward to write them in Cartesian Kerr-Schild coordinates. The
128  * coordinate transformation in gr::KerrSchildCoords helps read the Jacobian
129  * matrix, which, applied to the azimuthal flow of the disk, gives
130  *
131  * \f{align*}
132  * u_\text{KS}^\mu = u^t(1, -y\Omega, x\Omega, 0),
133  * \f}
134  *
135  * where \f$u^t\f$ and \f$\Omega\f$ are now understood as functions of the
136  * Kerr-Schild coordinates. Finally, the spatial velocity can be readily
137  * obtained from its definition,
138  *
139  * \f{align*}
140  * \alpha v^i = \frac{u^i}{u^t} + \beta^i,
141  * \f}
142  *
143  * where \f$\alpha\f$ and \f$\beta^i\f$ are the lapse and the shift,
144  * respectively.
145  *
146  * \note Kozlowski et al. \cite Kozlowski1978aa denote
147  * \f$l_* = u_\phi u^t\f$ in order to
148  * distinguish this quantity from their own definition \f$l = - u_\phi/u_t\f$.
149  */

366  // Grab the metric variables
367  template <typename DataType, typename Tag,
368  Requires<not tmpl::list_contains_v<hydro::grmhd_tags<DataType>,
369  Tag>> = nullptr>
370  tuples::TaggedTuple<Tag> variables(
371  const tnsr::I<DataType, 3>& /*x*/, tmpl::list<Tag> /*meta*/,
372  IntermediateVariables<DataType, true>& vars, const size_t index) const
373  noexcept {
374  // The only hydro variables that use the GR solution are spatial velocity
375  // and Lorentz factor. If those have already been set (their index in the
376  // typelist is lower than our index) then we can safely std::move the GR
377  // solution out, assuming nobody gets the same variable twice (e.g. trying
378  // to retrieve the lapse twice from the solution in the same call to the
379  // function), but then TaggedTuple would explode horribly, so nothing to
380  // worry about.
381  // Analytic solutions are used in Dirichlet boundary conditions and thus are
382  // called quite frequently, making some optimization worthwhile.
383  if (index > vars.spatial_velocity_index and
384  index > vars.lorentz_factor_index) {
385  return {std::move(get<Tag>(vars.kerr_schild_soln))};
386  }
387  return {get<Tag>(vars.kerr_schild_soln)};
388  }
