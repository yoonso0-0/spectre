Update the boundary_data_variables entries for all tags in Tags::characteristic_worldtube_boundary_tags to the boundary data from the analytic solution at time.

Details

This class retrieves metric boundary data from the Cce::Solutions::WorldtubeData derived class that represents an analytic solution, then dispatches to Cce::create_bondi_boundary_data() to construct the Bondi values into the provided Variables
