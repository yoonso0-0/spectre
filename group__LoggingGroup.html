std::string info_from_build ( )

Information about the version, date, host, git commit, and link time.

The information returned by this function is invaluable for identifying the version of the code used in a simulation, as well as which host, the date the code was compiled, and the time of linkage.

We declare the function extern "C" so its symbol is not mangled and we can ignore that it is undefined until link time (see CMakeLists.txt). Note that the std::string return type is not compatible with C, but that's OK as long as we're not calling or defining the function in C. See Microsoft's C4190: https://docs.microsoft.com/en-us/cpp/error-messages/compiler-warnings/compiler-warning-level-1-c4190
