134  * \ingroup SurfacesGroup
135  * \brief Extrinsic curvature of a 2D `Strahlkorper` embedded in a 3D space.
136  *
137  * \details Implements Eq. (D.43) of Carroll's Spacetime and Geometry text.
138  * Specifically,
139  * \f$ K_{ij} = P^k_i P^l_j \nabla_{(k} S_{l)} \f$, where
140  * \f$ P^k_i = \delta^k_i - S^k S_i \f$,
141  * `grad_normal` is the quantity \f$ \nabla_k S_l \f$ returned by
142  * `StrahlkorperGr::grad_unit_normal_one_form`, and `unit_normal_vector` is
143  * \f$S^i = g^{ij} S_j\f$ where \f$S_j\f$ is the unit normal one form.
144  * Not to be confused with the extrinsic curvature of a 3D spatial slice
145  * embedded in 3+1 spacetime.
146  * Because StrahlkorperGr::grad_unit_normal_one_form is symmetric, this
147  * can be expanded into
148  * \f$ K_{ij} = \nabla_{i}S_{j} - 2 S^k S_{(i}\nabla_{j)}S_k
149  * + S_i S_j S^k S^l \nabla_{k} n_{l}\f$.
150  */
151 template <typename Frame>
152 void extrinsic_curvature(
153  gsl::not_null<tnsr::ii<DataVector, 3, Frame>*> result,
154  const tnsr::ii<DataVector, 3, Frame>& grad_normal,
155  const tnsr::i<DataVector, 3, Frame>& unit_normal_one_form,
156  const tnsr::I<DataVector, 3, Frame>& unit_normal_vector) noexcept;
157

167 /// \brief Intrinsic Ricci scalar of a 2D `Strahlkorper`.
168 ///
169 /// \details Implements Eq. (D.51) of
170 /// Sean Carroll's Spacetime and Geometry textbook (except correcting
171 /// sign errors: both extrinsic curvature terms are off by a minus sign
172 /// in Carroll's text but correct in Carroll's errata).
173 /// \f$ \hat{R}=R - 2 R_{ij} S^i S^j + K^2-K^{ij}K_{ij}.\f$
174 /// Here \f$\hat{R}\f$ is the intrinsic Ricci scalar curvature of
175 /// the Strahlkorper, \f$R\f$ and \f$R_{ij}\f$ are the Ricci scalar and
176 /// Ricci tensor of the 3D space that contains the Strahlkorper,
177 /// \f$ K_{ij} \f$ the output of StrahlkorperGr::extrinsic_curvature,
178 /// \f$ K \f$ is the trace of \f$K_{ij}\f$,
179 /// and `unit_normal_vector` is
180 /// \f$S^i = g^{ij} S_j\f$ where \f$S_j\f$ is the unit normal one form.
181 template <typename Frame>
182 void ricci_scalar(
183  gsl::not_null<Scalar<DataVector>*> result,
184  const tnsr::ii<DataVector, 3, Frame>& spatial_ricci_tensor,
185  const tnsr::I<DataVector, 3, Frame>& unit_normal_vector,
186  const tnsr::ii<DataVector, 3, Frame>& extrinsic_curvature,
187  const tnsr::II<DataVector, 3, Frame>& upper_spatial_metric) noexcept;

198 /*!
199  * \ingroup SurfacesGroup
200  * \brief Area element of a 2D `Strahlkorper`.
201  *
202  * \details Implements Eq. (D.13), using Eqs. (D.4) and (D.5),
203  * of \cite Baumgarte1996hh. Specifically, computes
204  * \f$\sqrt{(\Theta^i\Theta_i)(\Phi^j\Phi_j)-(\Theta^i\Phi_i)^2}\f$,
205  * \f$\Theta^i=\left(n^i(n_j-s_j) r J^j_\theta + r J^i_\theta\right)\f$,
206  * \f$\Phi^i=\left(n^i(n_j-s_j)r J^j_\phi + r J^i_\phi\right)\f$,
207  * and \f$\Theta^i\f$ and \f$\Phi^i\f$ are lowered by the
208  * 3D spatial metric \f$g_{ij}\f$. Here \f$J^i_\alpha\f$, \f$s_j\f$,
209  * \f$r\f$, and \f$n^i=n_i\f$ correspond to the input arguments
210  * `jacobian`, `normal_one_form`, `radius`, and `r_hat`, respectively;
211  * these input arguments depend only on the Strahlkorper, not on the
212  * metric, and can be computed from a Strahlkorper using ComputeItems
213  * in `StrahlkorperTags`. Note that this does not include the factor
214  * of \f$\sin\theta\f$, i.e., this returns \f$r^2\f$ for a spherical
215  * `Strahlkorper` in flat space.
216  * This choice makes the area element returned here compatible with
217  * `definite_integral` defined in `YlmSpherePack.hpp`.
218  */
219 template <typename Frame>
220 void area_element(gsl::not_null<Scalar<DataVector>*> result,
221  const tnsr::ii<DataVector, 3, Frame>& spatial_metric,
222  const StrahlkorperTags::aliases::Jacobian<Frame>& jacobian,
223  const tnsr::i<DataVector, 3, Frame>& normal_one_form,
224  const DataVector& radius,
225  const tnsr::i<DataVector, 3, Frame>& r_hat) noexcept;
226 
227 template <typename Frame>
228 Scalar<DataVector> area_element(

312 /*!
313  * \ingroup SurfacesGroup
314  * \brief Spin function of a 2D `Strahlkorper`.
315  *
316  * \details See Eqs. (2) and (10)
317  * of \cite Owen2017yaj. This function computes the
318  * "spin function," which is an ingredient for horizon surface integrals that
319  * measure quasilocal spin. This function is proportional to the imaginary part
320  * of the horizon's complex scalar curvature. For Kerr black holes, the spin
321  * function is proportional to the horizon vorticity. It is also useful for
322  * visualizing the direction of a black hole's spin.
323  * Specifically, this function computes
324  * \f$\Omega = \epsilon^{AB}\nabla_A\omega_B\f$,
325  * where capital indices index the tangent bundle of the surface and
326  * where \f$\omega_\mu=(K_{\rho\nu}-K g_{\rho\nu})h_\mu^\rho s^\nu\f$ is
327  * the curl of the angular momentum density of the surface,
328  * \f$h^\rho_\mu = \delta_\mu^\rho + u_\mu u^\rho - s_\mu s^\rho\f$
329  * is the projector tangent to the 2-surface,
330  * \f$g_{\rho\nu} = \psi_{\rho\nu} + u_\rho u_\nu\f$ is the spatial
331  * metric of the spatial slice, \f$u^\rho\f$ is the unit normal to the
332  * spatial slice, and \f$s^\nu\f$ is the unit normal vector to the surface.
333  * Because the tangent basis vectors \f$e_A^\mu\f$ are
334  * orthogonal to both \f$u^\mu\f$ and \f$s^\mu\f$, it is straightforward
335  * to show that \f$\Omega = \epsilon^{AB} \nabla_A K_{B\mu}s^\mu\f$.
336  * This function uses the tangent vectors of the `Strahlkorper` to
337  * compute \f$K_{B\mu}s^\mu\f$ and then numerically computes the
338  * components of its gradient. The argument `area_element`
339  * can be computed via `StrahlkorperGr::area_element`.
340  * The argument `unit_normal_vector` can be found by raising the
341  * index of the one-form returned by `StrahlkorperGr::unit_normal_oneform`.
342  * The argument `tangents` is a Tangents that can be obtained from the
343  * StrahlkorperDataBox using the `StrahlkorperTags::Tangents` tag.
344  */
345 template <typename Frame>
346 void spin_function(
347  gsl::not_null<Scalar<DataVector>*> result,
348  const StrahlkorperTags::aliases::Jacobian<Frame>& tangents,
349  const Strahlkorper<Frame>& strahlkorper,
350  const tnsr::I<DataVector, 3, Frame>& unit_normal_vector,
351  const Scalar<DataVector>& area_element,
352  const tnsr::ii<DataVector, 3, Frame>& extrinsic_curvature) noexcept;
353 
354 template <typename Frame>
355 Scalar<DataVector> spin_function(
356  const StrahlkorperTags::aliases::Jacobian<Frame>& tangents,

363 /*!
364  * \ingroup SurfacesGroup
365  * \brief Spin magnitude measured on a 2D `Strahlkorper`.
366  *
367  * \details Measures the quasilocal spin magnitude of a Strahlkorper, by
368  * inserting \f$\alpha=1\f$ into Eq. (10) of \cite Owen2009sb
369  * and dividing by \f$8\pi\f$ to yield the spin magnitude. The
370  * spin magnitude is a Euclidean norm of surface integrals over the horizon
371  * \f$S = \frac{1}{8\pi}\oint z \Omega dA\f$,
372  * where \f$\Omega\f$ is obtained via `StrahlkorperGr::spin_function()`,
373  * \f$dA\f$ is the area element, and \f$z\f$ (the "spin potential") is a
374  * solution of a generalized eigenproblem given by Eq. (9) of
375  * \cite Owen2009sb. Specifically,
376  * \f$\nabla^4 z + \nabla \cdot (R\nabla z) = \lambda \nabla^2 z\f$, where
377  * \f$R\f$ is obtained via `StrahlkorperGr::ricci_scalar()`. The spin
378  * magnitude is the Euclidean norm of the three values of \f$S\f$ obtained from
379  * the eigenvectors \f$z\f$ with the 3 smallest-magnitude
380  * eigenvalues \f$\lambda\f$. Note that this formulation of the eigenproblem
381  * uses the "Owen" normalization, Eq. (A9) and surrounding discussion in
382  * \cite Lovelace2008tw.
383  * The eigenvectors are normalized with the "Kerr normalization",
384  * Eq. (A22) of \cite Lovelace2008tw.
385  * The argument `spatial_metric` is the metric of the 3D spatial slice
386  * evaluated on the `Strahlkorper`.
387  * The argument `tangents` can be obtained from the StrahlkorperDataBox
388  * using the `StrahlkorperTags::Tangents` tag, and the argument
389  * `unit_normal_vector` can
390  * be found by raising the index of the one-form returned by
391  * `StrahlkorperGr::unit_normal_one_form`.
392  * The argument `ylm` is the `YlmSpherepack` of the `Strahlkorper`.
393  * The argument `area_element`
394  * can be computed via `StrahlkorperGr::area_element`.
395  */
396 template <typename Frame>
397 double dimensionful_spin_magnitude(
398  const Scalar<DataVector>& ricci_scalar,
399  const Scalar<DataVector>& spin_function,
400  const tnsr::ii<DataVector, 3, Frame>& spatial_metric,
401  const StrahlkorperTags::aliases::Jacobian<Frame>& tangents,
402  const YlmSpherepack& ylm, const Scalar<DataVector>& area_element) noexcept;

404 /*!
405  * \ingroup SurfacesGroup
406  * \brief Spin vector of a 2D `Strahlkorper`.
407  *
408  * \details Computes the spin vector of a `Strahlkorper` in a `Frame`, such as
409  * `Frame::Inertial`. The result is a `std::array<double, 3>` containing the
410  * Cartesian components of the spin vector, whose magnitude is
411  * `spin_magnitude`. This function will return the dimensionless spin
412  * components if `spin_magnitude` is the dimensionless spin magnitude, and
413  * it will return the dimensionful spin components if `spin_magnitude` is
414  * the dimensionful spin magnitude. The spin vector is given by
415  * a surface integral over the horizon \f$\mathcal{H}\f$ [Eq. (25) of
416  * \cite Owen2017yaj]:
417  * \f$S^i = \frac{S}{N} \oint_\mathcal{H} dA \Omega (x^i - x^i_0 - x^i_R) \f$,
418  * where \f$S\f$ is the spin magnitude,
419  * \f$N\f$ is a normalization factor enforcing \f$\delta_{ij}S^iS^j = S\f$,
420  * \f$dA\f$ is the area element (via `StrahlkorperGr::area_element`),
421  * \f$\Omega\f$ is the "spin function" (via `StrahlkorperGr::spin_function`),
422  * \f$x^i\f$ are the `Frame` coordinates of points on the `Strahlkorper`,
423  * \f$x^i_0\f$ are the `Frame` coordinates of the center of the Strahlkorper,
424  * \f$x^i_R = \frac{1}{8\pi}\oint_\mathcal{H} dA (x^i - x^i_0) R \f$,
425  * and \f$R\f$ is the intrinsic Ricci scalar of the `Strahlkorper`
426  * (via `StrahlkorperGr::ricci_scalar`).
427  * Note that measuring positions on the horizon relative to
428  * \f$x^i_0 + x^i_R\f$ instead of \f$x^i_0\f$ ensures that the mass dipole
429  * moment vanishes. Also note that \f$x^i - x^i_0\f$ is
430  * is the product of `StrahlkorperTags::Rhat` and `StrahlkorperTags::Radius`.
431  */
432 template <typename Frame>
