Detailed Description

Analytic initial data for an advecting magnetic field loop.

This test, originally proposed in [29] and presented in a slightly modified form by [53], a region with annular cross section with the specified InnerRadius and OuterRadius is given a non-zero azimuthal magnetic field of constant magnitude MagFieldStrength with zero magnetic field outside the loop. Inside the InnerRadius the magnetic field strength falls to zero quadratically. The loop is embedded in an ideal fluid with the given AdiabaticIndex, RestMassDensity and Pressure with a uniform AdvectionVelocity. The magnetic field loop should advect across the grid, maintaining its shape and strength, as long as the magnetic pressure is negligible compared to the thermal pressure.

This test diagnoses how well the evolution scheme preserves the no-monopole condition, as well as the diffusivity of the scheme.

The standard test setup is done on \(x \in [-1,1]\), \(y \in [-0.5, 0.5]\), with periodic boundary conditions and with the following values given for the options:

  • InnerRadius: 0.06
  • OuterRadius: 0.3
  • RestMassDensity: 1.0
  • Pressure: 1.0
  • AdvectionVelocity: [0.08164965809277261, 0.040824829046386304, 0.0]
  • MagFieldStrength: 0.001
  • AdiabaticIndex: 1.66666666666666667
