Detailed Description

template<typename ScriValuesToObserve>
struct Cce::Actions::InitializeCharacteristicEvolutionScri< ScriValuesToObserve >

Initializes the CharacteristicEvolution component with contents needed to perform the interpolation at scri+.

Details

Sets up the DataBox to be ready to store data in the scri+ interpolators and perform interpolation for the final scri+ outputs.

DataBox changes:

  • Modifies: nothing
  • Adds:
    • Cce::Tags::InterpolationManager<ComplexDataVector, Tag> for each Tag in scri_values_to_observe
  • Removes: nothing
Note
This action relies on the SetupDataBox aggregated initialization mechanism, so Actions::SetupDataBox must be present in the Initialization phase action list prior to this action.
