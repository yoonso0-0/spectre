Detailed Description

template<typename FluxesComputerType>
struct elliptic::Tags::FluxesComputer< FluxesComputerType >

Holds an object that computes the principal part of the elliptic PDEs.

Details

The FluxesComputerType must have an apply function that computes fluxes from the system fields.

When placed in the cache, the FluxesComputerType is default-constructed. Provide tags that derive from this tag to construct it differently, e.g. to construct it from problem-specific options or retrieve it from an analytic solution.
